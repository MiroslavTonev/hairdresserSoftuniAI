### Create a plan for a project

### Purpose
The project is about booking for hairdresser's. It is going to use openfreemap.org embeded.

### Main functionallity:
There will be two main roles. Hairdresser onwer which will mark their location on the map, their working hours and working days and clients who will be able to book free hours on particular hairdresser salon.
For now keep it simple with only single owner with ability to select his miltiple locations

When you click on the map and select particular salon it will be shown a page with some calendar to be able to chose which hour the client wants to book

### Languages and frameworks.
The backend part will be written in java with simple database. Spring will be optional but if it is necessary it can be used.
The frontend will be react where it is necessary. If it is possible, use vanila JS
Keep it simple as it can, the backend could be deployd on embeded server like tomcat or similar. No docker or anything else, it must be run localy.

### Future functionalities
There will be some authentication, for now the role could be just selected, later after loging it will be implemented the ability for multiple owners.

On next stage the owner could remove a booking made by it's clients. It will have ability for notes from owners and clients. The owner will be able to make a note for example that this hour would be possible not available later. The client will have the ability to make a note for example that he expects something specific and to let know the owner.
